# KTwo Dark Mode Toggle

A WordPress plugin that adds a sleek dark mode toggle button to your KTwo WordPress theme.  

## Features
- Toggle between dark and light modes.
- Saves user preferences using localStorage.
- Easy integration with your WordPress theme.

## Installation
1. Download the plugin zip file or clone the repository.
2. Upload the plugin folder to `wp-content/plugins/`.
3. Activate the plugin from the WordPress Admin Dashboard.

## Usage
- The dark mode toggle button is automatically added to your site's footer.
- Customize styles by editing the `assets/css/dark-mode.css`.
